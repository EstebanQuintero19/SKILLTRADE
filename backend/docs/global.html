Usuario

Esquema de usuario en SkillTrade Representa a un usuario registrado en la plataforma, incluyendo datos personales, autenticación, visibilidad, estadísticas y métodos de utilidad.
Type:
  • Object
Properties:
Name Type Attributes Description
nombre string Nombre del usuario (2-50 caracteres, requerido).
email string Correo electrónico único y válido (requerido).
password string Contraseña del usuario (mínimo 6 caracteres, requerida).
apiKey string <optional>
 Clave de acceso API generada automáticamente.
foto string <optional>
 URL o ruta local de la foto de perfil.
biografia string <optional>
 Breve descripción (máximo 500 caracteres).
telefono string <optional>
 Número de teléfono en formato internacional.
visibilidad 'publico' | 'privado' Controla la visibilidad del perfil.
fechaCreacion Date Fecha en que se creó el usuario.
ultimoAcceso Date Última vez que accedió a la plataforma.
rol 'admin' | 'usuario' Rol del usuario (por defecto: 'usuario').
activo boolean Indica si el usuario está activo.
estadisticas Object Datos de actividad del usuario.
Properties
Name Type Description
cursosCreados number Cursos creados por el usuario.
intercambiosRealizados number Intercambios realizados.
suscripcionesActivas number Número de suscripciones activas.
